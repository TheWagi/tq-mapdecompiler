~~~~
Source
-->Levels
---->World
------>world01.map
------>world01.sd
------>world01.wrl
------>Babylon
------>Egypt
------>Greece
------>Olympus
------>Orient
-->XPack
~~~~

Usage:
~~~~
.\MAPDecomp world01.map world01.wrl
~~~~
